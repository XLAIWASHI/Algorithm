## 带权
**带权** 的实现思路很简单 ，永远将小树的挂在大树的下面，避免出现树深度过高的情况

## 带权代码

### 按大小合并
```cpp
//UF类
#include <vector>

class UF
{
public:
    UF(int n);
    ~UF();
    void unite(int x, int y);
    bool isConnected(int x, int y);
private:
    int root(int i);
    std::vector<int> id;
    std::vector<int> sz;
};
```

```cpp
//UF类实现
#include <iostream>
#include "UnionFind.h"

UF::UF(int n)
{
    id.resize(n);
    for(int i = 0; i < n; i++)
    {
        id[i] = i;
    }
    sz.resize(n, 1);
}

UF::~UF()
{
}

void UF::unite(int x, int y)
{
    int i = root(x);
    int j = root(y);
    if(i == j) //有sz数组，所以如果已经连接，就需要返回，否则sz数组会乱
    {
        return;
    }
    if(sz[i] < sz[j])
    {
        id[i] = j;
        sz[j] += sz[i];
    }
    else
    {
        id[j] = i;
        sz[i] += sz[j];
    }
}

bool UF::isConnected(int x, int y)
{
    return root(x) == root(y);
}

int UF::root(int i)
{
    while(i != id[i])
    {
        i = id[i];
    }
    return i;
}
```

## 讲解

两种方法，的最大层数都是  $\log_2 N$ ，下面会进行讲解
### 按大小合并
1. 首先，我们先假定我们当前的树是 **小树x** ；那么大树只能是 >= 小树的大小，才能使合并后的树深度 + 1；（因为，等于的时候，会创造出一个新的深度，而大于的时候，会使用大树的深度）
2. 那么令小树的大小为 s ，则 大树的大小为 >= s
3. (s + >= s) == (>= 2s)
4. 既，如果 x 的深度 + 1，合并后的树总大小 从s 变为了 **至少2s** 
5. 当深度为0时，节点至少为1；当深度为1时，节点至少为2；当深度为2时，节点至少为4；当深度为3时，节点至少为8；
6. 如果深度为d，节点至少为 $2^d$
7. 因为总节点数只有 N，所以 2^d <= N，所以 d <= $\log_2 N$
注：这里的“小树”和“大树”是动态比较出来的，如果当前树不是小数，那它就不需要被挂，身份会自动互换。
